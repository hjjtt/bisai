# CONTEXT.md

## 领域语言

- **实训任务 (TrainingTask)**: 教师创建的训练任务，学生提交文件
- **提交 (Submission)**: 学生对某个任务的文件提交
- **文件 (FileEntity)**: 学生上传的文档（PDF/Word/图片）
- **知识库 (Knowledge)**: 教师上传的参考资料，用于 RAG 检索增强
- **解析 (Parse)**: AI 提取文档内容
- **核查 (Check)**: AI 对比文档与知识库，生成核查结果
- **评分 (Score)**: AI/教师根据指标对提交打分
- **成绩发布**: 教师确认后向学生公开成绩

## 角色

- **学生**: 上传文件、查看成绩
- **教师**: 创建任务、触发 AI 分析、复核评分、发布成绩
- **管理员**: 用户管理、系统配置、全局监控

## 架构

Spring Boot 3 + Vue 3 单体应用，前端通过 Vite 代理访问后端 API。AI 能力通过 ModelScope 平台（Qwen 模型）提供。异步任务用数据库轮询（非消息队列）。
